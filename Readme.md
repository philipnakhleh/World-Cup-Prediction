# World Cup 2026 Winner Prediction

A machine learning project that predicts the FIFA World Cup 2026 winner using historical World Cup data, current FIFA rankings, and a Random Forest classifier.

## 📋 Overview

This project combines historical World Cup performance data with current FIFA world rankings to build a machine learning model that predicts which team will win the 2026 FIFA World Cup. The prediction leverages:

- Historical World Cup results (1994-2022)
- Current FIFA world rankings
- Historical team performance metrics

## 📁 Project Structure

- **WorldCup_2026_Winner_Prediction.ipynb** - Main Jupyter notebook containing the prediction model and analysis
- **extract_fifa_ranking.py** - Python script to extract current FIFA world rankings from HTML
- **world_cup_historical_data.csv** - Historical World Cup tournament data
- **world_cup_results_history.csv** - Historical World Cup match results
- **qualified_teams.csv** - Teams qualified for 2026 World Cup
- **Fifa_latest_ranking.csv** - Current FIFA world rankings (generated by extract_fifa_ranking.py)

## 🚀 Getting Started

### 1. Extract Current FIFA Rankings

The `extract_fifa_ranking.py` script extracts the latest FIFA world rankings from the official FIFA website.

**Prerequisite:** Download the FIFA world ranking page to HTML format from:
https://inside.fifa.com/fifa-world-ranking/men

**Usage:**
```bash
python extract_fifa_ranking.py --input ranking.html --output fifa_ranking.csv
```

This will generate `Fifa_latest_ranking.csv` with columns:
- `fifa_rank` - Current world ranking position
- `team` - Team name
- `country_code` - 3-letter country code
- `fifa_points` - Current FIFA ranking points
- `confederation` - Continental confederation (UEFA, CONMEBOL, etc.)

### 2. Run the Prediction Notebook

Open and run `WorldCup_2026_Winner_Prediction.ipynb` in Jupyter Notebook or VS Code:

```bash
jupyter notebook WorldCup_2026_Winner_Prediction.ipynb
```

The notebook performs the following steps:
1. Loads historical World Cup data and current FIFA rankings
2. Engineers features including team appearances, previous wins, confederation, and ranking
3. Trains a Random Forest Classifier using Leave-One-Group-Out cross-validation
4. Evaluates model performance
5. Generates predictions for 2026 World Cup qualified teams
6. Displays prediction results and statistics

## 📊 Model Features

The machine learning model uses the following features:

- **FIFA Rank** - Current world ranking position
- **FIFA Points** - Current ranking points
- **Confederation** - Continental confederation (encoded)
- **Previous Tournament Appearances** - Historical tournament participation count
- **Previous Wins** - Total historical World Cup wins
- **Home Advantage** - Whether team is a 2026 tournament host
